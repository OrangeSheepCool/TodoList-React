# REACT-TS-TODOLIST

### 功能描述

1. **添加待办事项**
  向输入框中输入内容后，按“回车”键或点击“添加”按钮，向列表中添加事项

2. **切换完成状态**
  点击单个事项的复选框，切换该事项的完成状态
  点击全选的复选框，切换全部事项的完成状态

3. **移除待办事项**
  <p>点击内容后的“移除”按钮，选择“确认”操作，从列表中移除该事项</p>

4. **移除已完成待办事项**
  点击“移除已完成的待办事项”字样，选择“确认”操作，从列表中移除已完成事项

5. **数据存储**
  初始化时会从本地存储中读取数据，数据改变时会更新本地存储
